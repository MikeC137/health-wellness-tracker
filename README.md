BMCC Student Wellness Tracker (C++)

A simple console program that lets students track their daily wellness habits such as mood, steps, and sleep. 
This project was created for an advanced programming techniques class and demonstrates core programming concepts like classes, vectors, sorting, file handling, 
and operator overloading.


* Features
  - Add a new wellness record
  - View all saved records (sorted by date & time)
  - Save records to a file
  - Load records from a file

* Concepts Used
  - Classes
  - Separate header & implementation files
  - Vectors of objects
  - File streams (ifstream, ofstream)
  - Operator overloading (operator<)
  - Friend function (printRecord)
  - Object sorting (manual bubble sort)
